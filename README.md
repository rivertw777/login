# 로그인 처리
## 로그인 요구사항
+ 홈 화면 - 로그인 전
  + 회원 가입
  + 로그인
+ 홈 화면 - 로그인 후
  + 본인 이름(환영 메시지)
  + 상품 관리
  + 로그 아웃
+ 보안 요구사항
  + 로그인 사용자만 상품에 접근하고, 관리할 수 있음
  + 로그인 하지 않은 사용자가 상품 관리에 접근하면 로그인 화면으로 이동
+ 회원 가입, 상품 관리

## 학습 내용
+ 로그인 처리(쿠키 사용)
+ 로그인 처리(세션 적용)
+ 서블릿 HTTP 세션
+ 필터, 인터셉터
  + 서블릿 필터
  + 스프링 인터셉터
  + ArgumentResolver 용용
